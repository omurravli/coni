# Coni_V02
Meet the Virtual Voice Asistant Coni

You need an OpenAI API to use this assistant.

## Setup for MacOS
```bash
git clone https://github.com/omurravli/coni.git
cd coni
bash install.sh
source .venv/bin/activate
add your api key to .env file as:
    OPENAI_API_KEY='YOUR_KEY'
python3 coni_v02.py
